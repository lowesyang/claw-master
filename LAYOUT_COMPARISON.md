## 5. 代码架构优化

### 状态管理
```typescript
// 新增平台状态管理
const [selectedPlatform, setSelectedPlatform] = useState<PlatformType>('all')

// 路由同步
useEffect(() => {
  if (location.pathname.startsWith('/clawnews')) {
    setSelectedPlatform('clawnews')
  } else if (location.pathname.startsWith('/moltbook')) {
    setSelectedPlatform('moltbook')
  } else if (location.pathname === '/') {
    setSelectedPlatform('all')
  }
}, [location.pathname])
```

### 动态菜单过滤
```typescript
// 根据选择的平台过滤菜单
const filteredSections = navSections.filter(section => {
  if (!section.platform) return true // 首页始终显示
  if (selectedPlatform === 'all') return false // "所有平台"时隐藏平台菜单
  return section.platform === selectedPlatform // 只显示选中的平台
})
```

### 平台切换处理
```typescript
const handlePlatformChange = (platform: PlatformType) => {
  setSelectedPlatform(platform)
  
  // 导航到对应页面
  if (platform === 'moltbook') {
    navigate('/moltbook')
  } else if (platform === 'clawnews') {
    navigate('/clawnews')
  } else {
    navigate('/')
  }
}
```

## 总结

本次优化显著提升了用户体验：

1. **首页**：从简单的平台选择升级为完整的产品介绍页
2. **侧边栏**：新增平台选择器，实现动态菜单显示
3. **导航**：更清晰的信息层级，减少认知负担
4. **视觉**：平台差异化配色，品牌识别更强
5. **交互**：流畅的切换体验，自动同步状态

这些改进让 Claw Master 作为多平台管理工具的定位更加明确，使用体验更加流畅。
